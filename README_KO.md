# VeInvite MVP v0.1

VeBetterDAO 신규 사용자 온보딩과 검증된 추천을 위한 테스트넷 MVP 소스입니다.

## 현재 구현된 것

- 추천인은 동시에 한 명만 초대
- 초대 링크 생성·복사·공유
- 링크를 처음 정상 연결한 한 명에게 귀속
- 지갑 연결 UI(VeChain Kit)
- 기존 사용자·다른 추천인·자기 추천·검토 중 예외 구조
- 초대 수락 대기 / 활성화 진행 / 완료 / 취소 상태
- 초대 취소 시 슬롯 복구 및 보상 자격 소멸
- 초대받은 사람의 3단계 미션 UI
- 서버 내부에서는 5개 활성화 조건 검증 구조
- 주간 비례 보상 계산 모듈
- Creator’s NFT 신청용 문안과 제출 체크리스트

## 실행 방법

1. Node.js 20 이상과 Yarn 또는 npm을 설치합니다.
2. `.env.example`을 `.env.local`로 복사합니다.
3. 아래 명령을 실행합니다.

```bash
npm install
npm run dev
```

4. 브라우저에서 `http://localhost:3000`을 엽니다.

기본값은 `NEXT_PUBLIC_DEMO_MODE=true`이므로 실제 지갑 없이도 데모 흐름을 확인할 수 있습니다.

## 반드시 교체해야 하는 데모 부분

현재 API는 메모리에만 저장되어 서버를 재시작하면 초기화됩니다. 실제 테스트넷 배포 전에는 Postgres/Supabase 등의 데이터베이스로 교체하고, 초대 코드 최초 1인 귀속을 데이터베이스 트랜잭션과 고유 제약으로 보장해야 합니다.

VeBetterDAO 활동 검증 또한 현재 데모입니다. 실제로는 다음을 공식 데이터나 온체인 이벤트로 확인해야 합니다.

- 서로 다른 VeBetter 앱 3개 사용
- B3TR 획득
- B3TR → VOT3 전환
- 투표 완료
- 기존 VeBetterDAO 사용 여부
- VePassport 또는 이에 준하는 부정 참여 위험 신호
